# 🛒 Blinkit Sales Dashboard | Power BI  

## 📌 Overview  
Interactive **Power BI dashboard** to analyze Blinkit sales data, providing insights into sales performance, item distribution, and outlet efficiency.

## 📊 Key Metrics  
- 💰 Total Sales: $1.2M+  
- 📦 Items: 8,500+  
- ⭐ Avg Rating: 3.9  

## 🚀 Features  
- KPI cards for quick insights  
- Filters for outlet location, size, and item type  
- Sales trend analysis (year-wise)  
- Category and outlet performance comparison  

## 🛠️ Tools Used  
- Power BI  
- Power Query  
- DAX  

## 🔍 Insights  
- Tier 3 outlets lead in sales  
- Medium outlets contribute highest revenue  
- Top categories: fruits, snacks, household items  

## ▶️ Usage  
Download `.pbix` → Open in Power BI → Explore using filters
